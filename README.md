This project is a command-line File Backup Tool implemented in Java. 
It demonstrates the use of Data Structures including:

• Stack – Backup History
• HashMap – File Indexing
• HashSet – Duplicate Path Prevention
• Recursive DFS – Directory Traversal

## Features
✔ Backup single files
✔ Backup full directories recursively
✔ Maintain backup history
✔ Maintain file index with timestamps
✔ Simple menu driven interface

## Technologies Used
Language : Java  
IDE      : IntelliJ IDEA  
OS       : Windows
